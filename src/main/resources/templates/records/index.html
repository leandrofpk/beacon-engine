Inmetro's Randomness Beacon

The randomness beacon is part of the Inmetro's effort SICCCiber project for disseminating reliable and public sources of randomness.
Currently, it are available by using a simple photo quantum true random number generator. However, the major goal of this project is the definition and construction of a system based on quantum entanglement for generating the random numbers.
PLEASE, DO NOT USE THE GENERATED VALUES AS CRYPTOGRAPHIC KEYS.

Notice that this is the first beta release of Inmetro's Beacon Service. More details about this project can be seen in SICCCiber. Contact us at

The dissemination of random number is supported by exposing a set of operations in a API. The currently implemented calls are listed below. Users submitting a request need to provide the pulse generation time in POSIX format (number of seconds since midnight UTC, January 1, 1970 (see http://en.wikipedia.org/wiki/Unix_time for more information and http://www.epochconverter.com for an online timestamp converter.) GET: /rest/record/< timestamp >
GET: rest/record/previous/< timestamp >
GET: rest/record/next/< timestamp >
GET: rest/record/last
Exemplo: /rest/record/last

GET: rest/record/index/
Exemplo: /rest/record/index/1
